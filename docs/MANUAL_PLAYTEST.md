# Manual Playtest Protocol

자동 검증 이후 실제 기기와 게시된 비공개 서버에서 수행할 최종 플레이 테스트입니다.

## 1. 단일 플레이 UX

- 첫 입장 5초 안에 모모 튜토리얼이 열리는지 확인
- 튜토리얼 종료 후 중앙 보라색 포털을 즉시 찾을 수 있는지 확인
- 포털 참가/취소 안내가 정확한지 확인
- 출제자 단어 세 개가 화면 밖으로 넘치지 않는지 확인
- 시작 시 그림판이 열리고 `—`로 접은 뒤 `그림판 열기`로 복원되는지 확인
- 마우스와 터치에서 빠르게 곡선을 그려도 선이 끊기지 않는지 확인
- 세밀·마커·굵게·부드럽게 프리셋과 100%·70%·40% 불투명도가 반영되는지 확인
- 원형·사각형 브러시, 지우개, 실행 취소, 전체 지우기가 정상 동작하는지 확인
- 로비에서 그림 말풍선을 공유하면 머리 위에 10초간 표시되는지 확인
- 그림 말풍선 80획 제한, 12초 쿨다운, 신고 버튼을 확인
- 라운드 45초와 20초 구간에 점진적 영문 힌트가 표시되는지 확인
- 출제자가 한 획 이상 그린 뒤 `완성 제출`로 즉시 공개 단계로 넘어가는지 확인
- `더 많은 색`에서 40개 승인 색상이 큰 버튼으로 보이고 선택 색이 실제 선에 반영되는지 확인
- 연필·마커·붓·크레용이 서로 다른 굵기와 질감으로 그려지는지 확인
- 작은 화면에서 도구 이름이 한·영으로 읽히고 가로 스크롤로 모든 도구에 접근 가능한지 확인
- Show·건너뛰기·Block·신고 버튼에 마우스를 올리면 짧은 설명이 표시되는지 확인
- 그림 말풍선 팝업을 클릭할 때 뒤쪽 메인 그림판이나 버튼이 함께 눌리지 않는지 확인
- 신고 후 숨긴 캔버스가 다음 라운드에 다시 표시되는지 확인
- 5라운드 후 순위, 우승 보상, 재참가 안내가 보이는지 확인

## 2. 멀티플레이 8–12인

- 12명이 동시에 참가해 정확히 12명만 매치에 포함되는지 확인
- 출제자가 순서대로 교대되고 비출제자에게 정답 데이터가 노출되지 않는지 확인
- 한국어/영어 정답, 대소문자, 공백, 복수형을 확인
- 동일 사용자의 중복 건너뛰기 투표가 한 표로만 처리되는지 확인
- 출제자 이탈, 추측자 이탈, 재접속 후에도 매치가 계속되는지 확인
- 정답·우승·퀘스트·구매 보상이 중복 지급되지 않는지 확인
- 30분 동안 라운드 정지, 캔버스 불일치, 데이터 유실이 없는지 확인

## 3. 기기 행렬

| 환경 | 필수 확인 |
|---|---|
| Windows PC | 마우스 드로잉, 채팅 입력, 1920×1080 및 1366×768 |
| Android 저사양 | 터치 입력 지연, 발열, 프레임 저하, 작은 화면 HUD |
| iPhone | 세로 전환 방지, 안전 영역, 키보드가 추측창을 가리지 않는지 |
| 태블릿 | 긴 터치 드로잉, UI 배율, 팝업 크기 |

## 4. 출시 통과 기준

- 치명적 진행 차단 0건
- 데이터 유실 및 중복 보상 0건
- 서버/클라이언트 오류 로그 0건
- 저사양 모바일에서 입력이 끊기지 않고 플레이 가능
- 신고된 작품이 갤러리 승인 대기열에 저장되지 않음
- 모든 항목을 통과한 뒤에만 비공개 QA 빌드를 제한 공개 베타로 전환
