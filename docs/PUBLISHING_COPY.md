# Publishing Copy

## Name

Momo's Art Academy: Guess & Draw

## Korean name

모모의 아트 아카데미: 그림 퀴즈

## Description

Choose a prompt, draw with a moderated palette and brush presets, and race to guess your friends' artwork.
Complete missions, unlock cosmetic palettes, and react to moderator-curated gallery drawings.
Supports English and Korean on computer, phone, and tablet.

검증된 색상과 브러시로 단어를 표현하고 친구들의 그림을 빠르게 맞혀보세요.
미션을 완료하고 꾸미기 팔레트를 모으며 운영자가 승인한 갤러리 작품에 반응할 수 있습니다.
PC와 모바일, 한국어와 영어를 지원합니다.
