# Moderation Runbook

## 초기 설정

1. `src/shared/Config.luau`의 `ADMIN_USER_IDS`에 운영자 Roblox UserId를 숫자로 등록합니다.
2. 운영자는 게임 안의 갤러리 관리자 NPC를 사용해 승인 대기 작품을 확인합니다.
3. `Approve`는 시즌 갤러리에 공개하고 `Reject`는 대기 큐에서 제거합니다.

## 처리 원칙

- 신고된 작품은 승인 대기 큐에 들어가지 않습니다.
- 반응을 한 개 이상 받은 작품만 승인 대기로 저장합니다.
- 성적·혐오·폭력·개인정보·저작권 침해·괴롭힘 표현은 거절합니다.
- 판단이 어려운 작품은 승인하지 않고 내부 기록과 Roblox 신고 도구를 확인합니다.
- 운영자 승인 권한은 최소 인원에게만 부여하고 변경 이력을 Git으로 관리합니다.

## 사고 대응

- 악성 사용자는 Roblox 기본 차단/신고 기능과 게임 내 캔버스 숨김을 함께 사용합니다.
- 반복적인 RemoteEvent 도배는 서버 속도 제한으로 차단되며 신고 기록은
  `ArtAcademyModeration_v1` DataStore에 저장됩니다.
- 갤러리 오승인이 발견되면 `approved_featured` 관리 데이터를 제거하고 새 서버에서 확인합니다.
