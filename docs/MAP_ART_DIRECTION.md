# Map Art Direction

## 목표

모모의 아트 아카데미 로비를 “기능을 선택하는 빈 광장”에서 “기다리는 동안 구경하고,
친구와 머물고, 다음 행동을 즉시 이해하는 소셜 아카데미”로 업그레이드한다.

## 벤치마크에서 적용한 원칙

- **Speed Draw**: 그림·갤러리·도구처럼 핵심 기능을 가까운 거리에서 즉시 식별하게 한다.
- **Dress To Impress**: 중앙 무대를 가장 강한 랜드마크로 만들고 주변 활동이 무대를 향하게 한다.
- **Adopt Me**: 색과 형태가 다른 작은 건물, 나무, 넓은 길로 장소를 기억하게 한다.
- **Roblox 환경 문서**: 전역광은 해질녘 수준의 낮은 밝기로 유지하고 포털·부스에만 국소 조명을 사용한다.

## 공간 구성

- 단일 세계 중심점: `WORLD_CENTER = Vector3.new(0, 0, 4)`
- 메인 주축: 남쪽 스폰에서 중앙 포털로 이어지는 직선
- 보조축: 중앙광장을 가로지르는 동서축과 네 건물의 방사축
- 전체 부지: 약 190×170 studs
- 아카데미 중앙 광장: 약 120×104 studs
- 스폰에서 중앙 포털까지 직선 시야와 색상 인레이 길
- 네 기능 건물의 정면과 입구가 모두 중앙 플레이존을 향함
- 낮은 명도의 라벤더 그레이 중앙광장과 딥 블루그린 외곽으로 눈부심 억제
- 중앙광장 링과 네 개의 연결 보행로가 메인 플레이존의 위계를 강조
- 중앙 포털 주변 8개 착석 가능한 대기 벤치
- 튜토리얼·상점·미션·갤러리의 네 개 독립 건물
- 외곽 24개 기둥과 컬러 지붕 링으로 원형 아카데미 실루엣 형성
- 야외 갤러리 프레임, 화단, 나무, 페인트 조형물로 대기 중 탐색거리 제공

## 캐릭터 디자인 규칙

- 모든 NPC의 몸, 얼굴, 명찰, 소품은 중앙 포털을 바라보는 동일한 로컬 좌표계를 사용
- 큐레이터 모모: 붓과 스케치북
- 팔레트 상인: 손에 든 컬러 팔레트
- 미션 선생님: 클립보드와 아카데미 배지
- 갤러리 관리자: 미니 액자와 갤러리 열쇠
- 공통 아카데미 코트, 앞치마, 벨트, 신발, 머리와 베레모로 통일된 실루엣 유지

## 성능 원칙

- 외부 무료 모델이나 텍스처 없이 Roblox 기본 Part와 Material만 사용
- 그림자 광원은 포털과 각 건물 입구에 제한
- Bloom 강도를 낮게 유지하고 DepthOfField는 사용하지 않음
- 모바일에서도 한 화면에 중앙 포털 또는 색상 건물 하나가 랜드마크로 보이도록 구성
