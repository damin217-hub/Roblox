# Drawing Game System Benchmark

## 조사한 시스템

- **skribbl.io**: 출제 단어 선택, 속도 기반 추측 점수, 출제자 보상, 점진적 글자 힌트,
  방장·투표 기반 신고와 음소거 기능.
- **Speed Draw**: 제한 시간 테마 드로잉, 작품 순차 공개, 플레이어 평가, 결과 연출.
- **Gartic Phone**: 그림과 해석을 다음 플레이어에게 전달하고 마지막에 변화 과정을 공개하는
  비경쟁 소셜 드로잉 구조.
- **Draw It 계열**: 한 명의 출제자와 여러 추측자, 정답 채팅, 부적절한 출제자 신고.

## 현재 적용

- 빠른 정답일수록 높은 점수, 정답자가 많을수록 출제자 점수 증가
- 라운드 중반 첫 글자 힌트, 후반 추가 글자 힌트
- 출제자가 완성한 그림을 제한 시간 전에 직접 제출
- 작품 공개 반응과 운영자 승인 갤러리
- 신고·숨김·차단·건너뛰기 투표
- 로비에서 그린 미니 그림을 머리 위 말풍선으로 10초간 공유
- 그림 말풍선은 승인 팔레트, 최대 80획, 12초 재사용 대기시간, 신고 기능 적용

## 후속 후보

- 비공개 서버용 커스텀 단어장
- 캐주얼·팀전·그림 전달 모드
- 근접 정답 안내
- 매치 종료 시 작품 타임라인과 베스트 리액션 시상
